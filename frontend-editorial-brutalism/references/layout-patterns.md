# Layout Patterns Reference

## Pattern 1: Cinematic Hero + Minimal Utility Header

Use a restrained top bar with one strong primary action.
Pair a large typographic headline with a single high-art image plane.
Keep supporting copy short and position it as a secondary block.

## Pattern 2: Editorial Story Band

Alternate dense text bands with image-led breathing bands.
Use contrasting section tempos: calm, intense, calm.
Add subtle directional cues to guide scan order.

## Pattern 3: Product Authority Strip (Watches)

Use a high-contrast close-up image with technical spec microcopy.
Anchor one premium CTA near the spec block.
Keep the number of simultaneous product cards low.

## Pattern 4: Hospitality Experience Grid (Hotels)

Compose 2-3 experiential highlights instead of long amenity lists.
Show room, dining, and atmosphere as distinct visual chapters.
Preserve one direct booking path in every major viewport.

## Pattern 5: Magazine-Inspired Feature Stack

Start with a manifesto section, continue with feature essays, end with conversion.
Use pull quotes and captions as rhythm markers.
Avoid repetitive card grids unless required for catalog pages.
